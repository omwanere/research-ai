# AI Research Paper Assistant

An AI-powered platform to discover, understand, and explore research papers using modern AI architecture.

## Features

* Semantic research paper search
* AI-powered summaries
* Paper chat assistant
* Research recommendation engine
* Citation graph visualization

## Tech Stack

### AI

* Sentence Transformers
* BGE Embeddings
* Ollama
* RAG Architecture

### Backend

* FastAPI
* PostgreSQL
* Qdrant

### Frontend

* Next.js
* TailwindCSS
* Framer Motion
* React Three Fiber

## Architecture

User → Frontend → Backend → AI Engine → Vector Database

## Goal

Build a modern AI research assistant similar to Perplexity or Elicit.
